### 简单说明

#### lambda表达式
- lambda 表达式总是括在花括号中
- 参数声明放在花括号内
  - 并有可选的类型标注。 
- 函数体跟在一个 -> 之后。


#### lambda表达式 p01
- 简单的例子
- 注意：
  - 如果返回类型可以推断，可以不写
  - 如果入参类型可以推断，可以不写
  - 也可以用 invoke 写法

#### 传递末尾的 lambda 表达式 p01
- 对于高阶函数，末尾是函数的
- 末尾函数，可以用 lambda简化入参

#### 其他写法 
- it：单个参数的隐式名称
  - 只有1个入参
  - 可以省略入参，用it表示参数
- 下划线_ 用于未使用的变量
  - map.forEach { (_, value) -> println("$value!") }
