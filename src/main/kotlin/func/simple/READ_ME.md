### 简单说明


#### 函数 s01 （Simple）
- 定义函数，调用函数
- 入参，出参
- 默认参数
- 入参 qux: () -> Unit
  - 在 func 调用 qux()
  - 传参方式 【都是 qux 入参】
    - func(qux = {xxx}) 
    - func(1) {xxx}
    - func {xxx}


#### 函数 s02 （Simple）
- 可变实参 （可变参数）
  - vararg 关键字
    - fun foo(vararg strings: String)
    - foo(strings = *arrayOf("a", "b", "c"))
      - 入参前面添加 *
- 返回 Unit 的函数
  - 也就是 不需要返回值
    - return
    - return Unit
- 单表达式函数
  - 单个表达式构成时，可以省略花括号
    - 并且在 = 符号之后指定代码体
    - 可以推测返回值后， 可以省略 返回值
- 中缀表示法
  - 必须满足以下要求：
    - 它们必须是成员函数或扩展函数
    - 它们必须只有一个参数
    - 其参数不得接受可变数量的参数且不能有默认值。
  - 注意：
    - 用 this 接收原来的对象



#### 函数 s03 （Simple）
- 作用域相关
  - 局部函数
    - 一个函数，在另一个函数内部
  - 成员函数
    - 在类 或 对象 内部定义的函数
  - 泛型函数
    - 带泛型T的函数
  - 尾递归函数(略)
